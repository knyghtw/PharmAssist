# PharmAssist

## Deskripsi
PharmAssist adalah aplikasi manajemen apotek yang dapat membantu apoteker mengelola stok obat, penjualan, dan laporan dengan mudah dan efisien. Aplikasi ini dirancang menggunakan bahasa C# dan .NET Framework.

## Fitur
- Data Supplier
- Data Stok Obat
- Data Pembelian
- Data Penjualan
